
# Clash Schedule Bot

Clash Schedule Bot is a Discord bot that provides upcoming **League of Legends Clash tournament schedules** for the **EUW1 region**.

The bot retrieves Clash tournament information from the Riot Games API and posts schedule updates to Discord servers.

---

## How it works

The bot retrieves tournament schedule information from the Riot Games Clash API endpoint:

`/lol/clash/v1/tournaments`

Features:

- `/clash` command to display upcoming Clash tournaments

---

## Invite / Access

The bot is currently in **private use / testing** and is not publicly distributed.

Servers interested in testing the bot can contact the developer.

---

## Privacy Policy

Clash Schedule Bot stores only minimal operational data required to function, such as:

- Discord server ID
- Discord channel configuration
- cached Clash tournament schedule data

The bot **does not store Riot account information, personal user data, or authentication credentials**.

Stored configuration data is used only to deliver Clash schedule notifications.
